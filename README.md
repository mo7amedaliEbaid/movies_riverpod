# movies_riverpod
 
- Movies App.
- Clean Architecture.
- Riverpod State Management.
- Dependency injection.
- Multiple Themes.
- Bookmarks.
- Google Maps.
- Upcoming Screen.
- Home Screen.
- Movie Details Screen.
- Now Showing movies.
